# Hackathon Demo Scenario

## Credentials
- Admin: `admin@carbon.local` / `Admin123!`
- User: `user@carbon.local` / `User123!`

## Demo flow (8-10 minutes)
1. Login to web app.
2. Show dashboard KPIs and top emitting sites.
3. Open site list and inspect seeded sites.
4. Create a new site (quick form + materials).
5. Trigger carbon calculation and show result cards.
6. Re-run after editing the site and show history (`versionNo` increments).
7. Open comparison page and compare 2-3 sites.
8. Show mobile app login, site list, quick add, and result consultation.

## Messages for jury
- Construction and operation emissions are clearly split.
- Historical snapshots ensure auditability.
- Factor provider abstraction enables ADEME/API integration with minimal refactoring.
- Stack is Docker-ready and easy to industrialize.
